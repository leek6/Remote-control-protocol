# Remote-control-protocol

## Usage
### Input
In function packetize, change input method of incomming protocol.
Default is keyboard input.
``` Java
//---------------------------------------------------------------------------------- input
	input = scan.nextByte();	//test input: only ascii decimal
//----------------------------------------------------------------------------------
```

### Get info

``` Java
//------------------------------------------------------ get temperature & location when use
	byte currentTemp = 0;
	byte currentLoc = 0;
//------------------------------------------------------
```
